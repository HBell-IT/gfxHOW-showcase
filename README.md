# GfxHOW — Live Graphics & Church Presentation Platform

GfxHOW is a web-based live graphics and presentation system designed as a cost-effective alternative to tools like ProPresenter.

This project showcases a real-world system used in live church services to display announcements, song lyrics, and scripture in real time.

---

##  Key Features

*  Real-time graphics playout (Take, Next, Clear controls)
*  Template-driven rendering system (HTML/CSS + JSON)
*  Visual template designer for non-technical users
*  Scripture system with multi-language Bible support
*  Song and lyrics display system
*  Image and video asset support
*  Dynamic data binding using placeholders (e.g. `{{title}}`)
*  CSS animation lifecycle (entry, exit, transitions)
*  Event-driven messaging system for slide control
